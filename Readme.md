## Configuration in Claude Desktop

Add this to your claude ```claude_desktop_config.json``` :

```
{
    "mcpServers": {
        "mcp-server-winston-ai": {
            "command": "npx",
            "args": [
               "-y", "winston-ai-mcp"
            ],
            "env": {
                "WINSTONAI_API_KEY": "your-api-key"
            }
        }
    }
}
```

### Environnement Variables
